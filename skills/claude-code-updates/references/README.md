# References

This directory contains generated reference files from the cached Claude Code documentation.

Files in this directory are generated by the `/update-docs` command and should not be edited manually.

## Files

- `official-docs.md` - Parsed official documentation
- `releases.md` - Recent releases and changelog

## Regenerating

To regenerate these files, run:
```bash
cd ${CLAUDE_PLUGIN_ROOT} && bun run scripts/update-docs.ts --force
```
